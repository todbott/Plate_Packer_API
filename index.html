Part 1: The System

Iterative Plate Packer API

One of the two endpoints of this API (/plates) uses an iterative approach to solving the problem. It can be tested via the links below, or by navigating to the endpoint and adding the string to be tested as a url param called 'data'
Example: /plates?data=sanma-5:10
